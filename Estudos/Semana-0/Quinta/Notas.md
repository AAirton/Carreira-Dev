#Pensamento Computacional

## _Sequencia Logica de Passos Simples._


1.**Estrutura do algoritmo:**
- **estrutura do algoritmo** --> identação --> _sequência (passos objetivos)_:
       - linguagem natural (passo a passo)
	+ **uso de estados**
       - interpretável e executável por maquina; 
       - uso de entradas (vazão, modo) e parâmetros (ex. tamanho de alguma coisa, quantidade etc);
       - modularidade (organização em blocos); 
       - ling. especifica;
       - maquina executa o programa para resolver um problema especifico. 

- **estrutura do algoritmo** --> Tomada de decisões: 
       - condicionais (pergunta para decidir qual caminho seguir);
	+ sim ou não, por exemplo (variável booliana).
       - repetição de sequências  --> sequência de instruções (**loops** ou **laços**)
       
1.1.**Tipo de dados:**
- Texto; 
- Número inteiro;
- Numero decimal; 
- Booleano (T/f).

1.2.**Variável(=):**
- Armazena um conteúdo que poderá ser alterado:
	+ números, quantidades etc.
- Operações com variáveis:
	+ algoritmo par/impar (divisão por dois, restando ou não zero)
	+ algoritmo de soma (variável x,y,z ...)



2.**Operadores aritméticos**
- Impar/par (%2; restando ou não zero)
- Subtração (variável a,b,c --> c = a-b)

2.1. Operações relacionais:
- Igualdade (==): True/False 
- Diferente (!=): True/False
- Maior/Menor que (<,<=,>,>=): True/False

2.2. Operadores lógicos:
- E --> todas condições precisarão ser verdadeiras;
- OU --> pelo menos uma condição será verdadeira;
- NÃO --> inverter o valor logico.


3.Algoritmo (parte):
- Input (entrada);
- Processamento (logico aritmético)
- Output (saída)

4.**Funções:**
- Bloco de algoritmo que realiza uma tarefa especifica;
- Recebe entradas (parâmetros) e devolve saídas (retornos).
